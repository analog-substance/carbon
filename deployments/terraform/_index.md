## Prerequisites

In order to properly use Terraform, you'll need access to our AWS account.

## Installation

### MacOS

We can install terraform using Hashicorp's Brew Tap.

```bash
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
```
